# Настройка меню в IIKO

## Принцип работы

В выгрузке меню в IIKO создана группа "Меню сайта" с позициями, реализуемыми через сайт и мобильное приложение.

Группы и блюда из "Меню сайта" выгружаются в той же последовательности, что и в IIKO.

Названия, описания и фотографии групп и блюд на сайте и в приложение будут такие же, как в IIKO.

ФОТО

Если для сайта и приложения требуются служебные блюда, которые не должны отображаться в меню – например, платная доставка, акционные позиции или приборы для реализации в корзине, то эти блюда необходимо добавить в выгрузку в отдельную группу, вне группы "Меню сайта". 

## Структура меню

1. Создать в корне выгрузки группу "Меню сайта".

2. Выгрузить в неё группы с блюдами для сайта и приложения.

ФОТО

:::tip
Чтобы не нарушить работу действующего сайта, рекомендуем именно выгрузить повторно группы с блюдами, а не перемещать блюда, которые уже были в выгрузке
:::

3. При помощи стрелок настроить в группе "Меню сайта" последовательность групп и блюд, которая должна быть на сайте.

ФОТО 

## Описания и фотографии

:::tip
Рекомендации по фотографиям: JPG, до 1500 x 1500 px, до 500 Кб. 

Перед загрузкой необходимо сжать изображения через сервис compressjpg.com для ускорения загрузки на устройствах гостей. 
::: 

1. Дважды нажать ЛКМ по названию блюда, откроется карточка элемента меню.

2. Указать описание блюда в поле "Описание".

3. Привязать фотографию блюда.

4. При необходимости заменить "Название для сайта".

5.  Нажать кнопку "Сохранить и закрыть".

6.  Повторить для остальных блюд.

ФОТО

Таким же образом добавить описание и фотографии для всех блюд в "Меню доставки". 

Таким же образом добавить фотографии для групп, и при необходимости заменить названия групп.

## Выгрузка

1. Нажать кнопку "Выгрузить".

ФОТО

2. Перейти во вкладку "Состояния подключения".

ФОТО 

3. Нажать кнопку "Обновить".

ФОТО
